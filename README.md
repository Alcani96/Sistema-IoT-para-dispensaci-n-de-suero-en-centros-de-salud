# Sistema-IoT-para-dispensaci-n-de-suero-en-centros-de-salud
Repositorio del proyecto Sistema IoT para dispensación de suero en centros de salud del Máster Universitario en Internet De Las Cosas / Master in Internet of Things (IoT).

En este repositorio se puede encontrar una carpeta "Paciente 1" y el fichero Visualizacion.html

En la carpeta Paciente 1 se encuentran los 4 ficheros que permiten simular el sistema de dispensación de suero de un paciente a través de la aplicación Wokwi.
El fichero Visualizacion.html contiene el código necesario para visualizar el dashboard y el sistema de alarmas del sistema.
